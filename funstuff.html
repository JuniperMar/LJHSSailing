Educational Sailing Games

Linked here are some fun sailing-related games that will hopefully prove helpful for learning about the sport from sailing rules to boat designs.

Disclaimer: The advice and information given through these games is mostly for entertainment, and is not endorsed by LJHS or MBYC. While the information is intended to be as accurate as possible, please take it with a grain of salt since it was written by a current high school sailor, not a coach. Thanks! Enjoy the games!

Sailboat "2048"

Inspired by the classic game 2048, this version uses sailboats instead. Learn about different one-design fleets while playing this addictive game.

Parts of Sailboat Memory Game

Can you remember all the common parts of a sailboat? Sailing requires a lot of specific vocabulary to name different parts of the boat. Test your memory and how well you know your boat with this game.
